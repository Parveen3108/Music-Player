# Music Player

A modern React + Vite based Music Player application.

## Features
- Play / Pause music
- Next / Previous track
- Repeat & Shuffle
- Clean and responsive UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript

## Installation & Run
```bash
npm install
npm run dev
```

## Notes
- This repository was created from a React + Vite starter and customized for a music player app.
